### Задание 1

Дана схема для Cisco Packet Tracer, рассматриваемая на лекции.
На этой схеме уже настроено отслеживание интерфейсов маршрутизаторов Gi0/1 (для нулевой группы).
Аналогичным образом необходимо настроить отслеживание состояния интерфейсов Gi0/0 (для первой группы).
Чтобы проверить правильность настройки, отсоедините один из кабелей между одним из маршрутизаторов и коммутатором Switch0 и запустите пинг между PC0 и Server0.
Отправьте на проверку получившуюся схему в формате pkt и скриншот, на котором виден процесс настройки маршрутизатора.
